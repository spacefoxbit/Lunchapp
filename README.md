# Lunchapp

A simple React webapp to track payments between three people (K, A, H) with swipe gestures, inspired by Gmail's email list UI.

## Features
- List of three people: K, A, H
- Swipe left: Archive under "I paid"
- Swipe right: Archive under "They paid"
- Clean, minimal design
- No backend, local state only

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
